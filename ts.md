## TypeScript Basics

- Static type checker that compiles to JS
- `tsc` converts .ts → .js
- Browsers can't read TS, only JS
- Prefer type inference over annotation
- `any` disables type checking — avoid it
- `unknown` = safer `any`, forces type check
- Compile time errors (TS catches) vs Runtime errors (TS doesn't)
- Strict mode: enables strictNullChecks and more
